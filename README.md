This dataset contains 8732 labeled sound excerpts (<=4s) of urban sounds from 10 classes:

<li>car_horn
<li>children_playing
<li>dog_bark
<li>drilling
<li>engine_idling
<li>gun_shot
<li>jackhammer
<li>siren
<li>street_music
